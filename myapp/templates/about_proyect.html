El objetivo del proyecto fue desarrollar un sistema de detección de objetos en video que pudiera reconocer diferentes tipos de objetos en tiempo real. Para ello, se utilizó el framework de detección de objetos YOLOv5, que es una versión mejorada de YOLO que utiliza técnicas de deep learning para lograr una mayor precisión en la detección de objetos #Yolov5 #OpenCv #Django #Python

Se implementó el modelo de detección de objetos de YOLOv5 utilizando OpenCV para procesar el video y llevar a cabo la detección de objetos en tiempo real. Además, se utilizó Django como framework web para desarrollar una interfaz de usuario que permitiera a los usuarios visualizar los resultados de la detección de objetos en tiempo rea
